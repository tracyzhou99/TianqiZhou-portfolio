These are the sample codes for the Post-COVID Adaptations: Online vs In-Person Education project.

1. eda_data_selection_cleaning.ipynb: Filtered submissions and comments that related to online and in-person education using PySpark. Created dummy variables for further analysis.

2. eda_covid_trend: Visualized trends of COVID new cases and Reddit posts using PySpark, Pandas, and Plotly.

3. nlp_sentiment_tfidf: Carried out the sentiment and TF-IDF analyses using Spark NLP.

4. ml_score_predict: Implemented linear regression and random forest regression for submissions and comments score prediction. Performed 4 groups of hyperparameter tuning and evaluated RMSE and R^2.
